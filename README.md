### AAA Pattern

The AAA Pattern which stands for (Arrange, Act, Assert) is a pattern used for structuring tests.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
npm install
```

## Usage

```bash
npm run test
```
